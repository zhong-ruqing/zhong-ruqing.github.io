[Project] Adaptive Weaving

Seminar Project | In Collaboration With Weijie Fu, Zhiqi Lin | 2018


Weaving the pattern according to the environment feecback.
Concept of this project is to create a weaving pattern based on the lightness feedback from the environment.
In this project we use arduino to conrol KUKA Robot to strat scan. The light inflection detector is used to sense the lightness level of the each grid of a grid plate. The sensing value is then sent back to arduino board. The pattern of weaving is generated by our algorithm and the corresponding command is sent back to KUKA.
