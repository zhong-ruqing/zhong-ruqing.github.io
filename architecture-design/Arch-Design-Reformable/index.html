[Arch Design] Reformable

Undergraduate Design Project | Zhejiang University | Architecture | In Collaboration with Mengying Hu | 2015


From 2D to 3D.
This project is an experiment that explores how to generate three-dimentional space with two-dimentional elements. Concept is to use each pair of strip as an element. Each element has 3 control points that can influence its form. The whole structure is made by those single elements. The result is realized in digital software as well as in physical world with bamboo.
